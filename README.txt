== Description ==
Add to cart button extenstion like the Divi's call to action button with simple-product selection.
== Installation ==


1. Upload `woodivi.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. A new divi module will be added with name WooCommerce Product
